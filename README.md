# About 
Brainfuck interpreter written in c.

# Build 
Just run `make`

# Usage 
Run a file 
```
./bf <FILEPATH>
```

Run an inline command 
```
./bf -c <BFCOMMAND>
```

# Examples

```
./bf tests/hello_world.bf
```
